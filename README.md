# Markdown Preview

A simple Markdown preview website built with pure HTML, CSS, and JavaScript.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript

## Supported Markdown Syntax

The syntax parsing is implemented from scratch in `script.js` and consists of the following elements:

- **Headings**: `#` through `######` for H1–H6.
- **Emphasis**: `**bold**` for bold and `*italic*` for italic text.
- **Code**: `` `inline code` `` for inline and ` ```code block``` ` for multi-line code.
- **Links & Images**: `[text](url)` for hyperlinks and `![alt](url)` for images.
- **Blockquotes**: `> text` for quoted blocks.
- **Lists**: `- item` for unordered lists and `1. item` for ordered lists.
- **Horizontal Rule**: `---` for a divider line.
